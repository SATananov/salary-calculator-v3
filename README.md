# Salary Calculator v3

Просто проект за изчисляване на заплата (TypeScript + HTML/CSS).

Структура на проекта:

- `index.html` — основна HTML страница
- `src/` — TypeScript кода: `app.ts`, `calculator.ts`, `storage.ts`, `types.ts`, `ui.ts`
- `styles.css` — стилове

Как да използвате (локално):

1. Уверете се, че имате Node.js и Git инсталирани.
2. Инсталирайте зависимости: `npm install`.
3. Стартирайте локален сървър или отворете `index.html` в браузър.

Как да публикувате в GitHub:

- Инициализирайте локален репозиториум и направете commit (това вече е подготвено в проекта).
- Създайте публичен репозиторий в GitHub и го добавете като remote.
- Изпратете (`push`) вашия `main` клон и копирайте публичния линк.

Ако искате, мога да подготвя и автоматично да инициализирам git и да направя първия commit локално — потвърдете и ще го направя.
